# lenreg_ticket_bot

Telegram-бот для мониторинга доступных талонов (слотов) на портале zdrav.lenreg.ru.

## Функциональность

- Регистрация пациента по ФИО, дате рождения, псевдониму
- Просмотр специальностей и врачей в выбранной поликлинике
- Мониторинг слотов с уведомлениями о появлении/исчезновении
- Автоудаление старых сообщений (TTL)
- Команда `/status` — метрики здоровья бота (только для админов)
- Поддержка трёх типов поликлиник: взрослая, стоматологическая, детская

## Настройка

Создать `.env` на основе [`.env.example`](.env.example) и заполнить переменные:

- `BOT_TOKEN` — токен Telegram-бота
- `PROXY_URL` — URL прокси (опционально)
