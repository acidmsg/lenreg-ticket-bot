---
description: "Запуск всех тестов pytest, при падениях — автофикс в debug-режиме"
---

Запусти все тесты проекта командой:

```powershell
python -m pytest tests/ -v > .tmp_test_results.txt 2>&1
```

Прочитай `.tmp_test_results.txt`. Если все тесты прошли — сообщи об успехе и удали временный файл.

Если есть упавшие тесты — проанализируй каждый упавший тест, найди корневую причину в исходном коде и исправь её. После исправления запусти тесты повторно. Повторяй до тех пор, пока все тесты не пройдут.

После завершения обязательно удали `.tmp_test_results.txt`.
