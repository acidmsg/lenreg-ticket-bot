---
description: "Быстрая проверка: параллельный запуск lint и test"
---

Запусти параллельно две проверки, делегировав их в соответствующие режимы:

1. **Линтинг** — делегируй в code mode задачу, идентичную команде `/lint` (ruff --fix + format + mypy + markdownlint + prettier).
2. **Тесты** — делегируй в debug mode задачу, идентичную команде `/test` (pytest tests/, при падениях — автофикс).

Дождись результатов обеих подзадач. Собери сводный отчёт: что исправлено, какие ошибки остались, общий статус (OK / есть проблемы).
