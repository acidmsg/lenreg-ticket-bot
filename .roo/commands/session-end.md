Выполни протокол завершения сессии по правилам из `.roo/rules/workflow.md`:

1. Обнови `.roo/sessions/SESSION_LOG.md` — допиши запись о текущей сессии (дата, выполненные задачи, изменённые файлы, результаты тестов если были).
2. Перенеси предыдущую запись из `SESSION_LOG.md` в конец `.roo/sessions/SESSION_ARCHIVE.md` (под заголовок `---`).
3. Обнови `.roo/tasks/AGENT_TASKS.md` — удали выполненные задачи из таблицы и чек-листа (не зачёркивать, а полностью удалить строки).
4. Прогони markdownlint: `npx markdownlint "specs/**/*.md" ".roo/**/*.md" "*.md"` — исправь найденные ошибки.
5. Прогони prettier: `npx prettier --write "specs/**/*.md" ".roo/**/*.md" "*.md"`.
6. Выполни `git status` чтобы показать итоговый список изменённых файлов.
