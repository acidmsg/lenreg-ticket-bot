# База знаний API

Унифицированный формат: **Назначение** → **Метод/URL** → **Параметры** → **Ответ** → **CURL**.

- [check_patient](check_patient.md) — поиск пациента по ФИО и дате рождения
- [speciality_list](speciality_list.md) — список специальностей врачей
- [doctor_list](doctor_list.md) — список врачей по специальности
- [appointment_list](appointment_list.md) — слоты (талоны) для врача
- [clinic_list](clinic_list.md) — список поликлиник по району
