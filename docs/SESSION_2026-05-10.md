### Проблема 2 — квадратные скобки в .env

`ADMIN_IDS: list[int]` требовал JSON-формата (`[123456789]`), что неочевидно.

**Исправление:** тип изменён на `str`, парсинг через `split(",")`:

```python
# config.py
ADMIN_IDS: str = ""

# handlers/common.py
admin_ids = [int(x.strip()) for x in settings.ADMIN_IDS.split(",") if x.strip()]
```

Теперь в `.env` единообразно со всеми ID:

```
ADMIN_IDS=120649243
ADMIN_IDS=120649243,987654321  # несколько через запятую
```
