# Изображения для сообщений Telegram-бота

Изображения используются как иллюстрации-заголовки в сообщениях бота (картинка сверху, текст снизу). Отправляются через метод `send_photo()` Telegram Bot API с параметрами `photo` и `caption`.

## Директория

Все изображения размещаются в [`src/assets/images/`](src/assets/images/). Поддерживается только формат **PNG**.

## Система именования

Имя файла строится по шаблону: **`{контекст}_{состояние}.png`**

| Компонент   | Тип          | Описание                                                              |
| ----------- | ------------ | --------------------------------------------------------------------- |
| `контекст`  | `snake_case` | Часть интерфейса: `patient`, `clinic`, `doctor_*`, `slot`             |
| `состояние` | `snake_case` | Ситуация/действие: `select`, `empty`, `available`, `new`, `decreased` |

### Правила

1. Имя файла — **строчные буквы**, слова разделены `_` (snake_case).
2. **Только PNG** (расширение `.png`).
3. Контекст и состояние разделены **ровно одним** символом `_`.
4. Составной контекст (например, `doctor_adult`) уже содержит `_`, дополнительный разделитель не нужен — имя читается как `{контекст}_{состояние}`, где `контекст` может быть составным.
5. Каждое изображение — **ровно один файл** (никаких вариантов через суффиксы `@2x` — при необходимости добавить отдельный `состояние`).

## Использование в коде

Изображения загружаются через `aiogram.types.FSInputFile`:

```python
from aiogram.types import FSInputFile
from pathlib import Path

ASSETS_DIR = Path(__file__).parent.parent / "assets" / "images"

# Отправка изображения с подписью
photo = FSInputFile(ASSETS_DIR / "slot_empty.png")
await bot.send_photo(
    chat_id=uid,
    photo=photo,
    caption="🧑‍⚕️ Иванов И.И.\nНомерков пока нет 🤷‍♂️",
)
```

Либо через вспомогательный словарь-маппинг (будет создан на этапе реализации):

```python
NOTIFY_IMAGES: dict[str, str] = {
    "empty": "slot_empty.png",
    "available": "slot_available.png",
    "new": "slot_new.png",
    "decreased": "slot_decreased.png",
}
```

## Требования к файлам

- Разрешение: **рекомендуется 800×400 px** (хорошо смотрится в мобильном Telegram).
- Размер: **не более 512 КБ** (Telegram сжимает, но лучше держать в разумных пределах).
- Фон: прозрачный или совпадающий с тёмной/светлой темой Telegram (по умолчанию — белый/светлый).
- Текст на изображении: **только декоративный**; основной текст всегда в `caption`.
